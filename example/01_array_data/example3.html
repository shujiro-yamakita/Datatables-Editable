DataTable-Editable - Example #3

ColumnDefs options with array-based datatable rows

Set the columnDefs option to change the HTML input type for each column. In this table, in the Extn. Column, the type attribute of the input tag displayed at the time of editing is number, and in the Start date column, the type attribute is date. If you want to specify how to display the date when editing the date, add 'format'. (By default, it is displayed as 'Y-m-d')
