DataTable-Editable - Example #13

Validate Draw

In manual mode, the table is not updated when you edit the cell.
However, if you have a cell whose value depends on the value of another cell, that can be a problem.
In that case, set the "row" or "table" option to the "validateDraw" option.
If "row" is set, the row will be recalculated and displayed when the cell is edited.
If "table" is set, the entire table will be recalculated and displayed.
